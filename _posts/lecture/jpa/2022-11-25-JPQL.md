# JPQL

- Java Persistence Query Language
- JPA는 다양한 쿼리 방법을 지원한다.
  - Native SQL
  - 쿼리 빌더 (Cirteria, Querydsl)
  - JDBC API
  - MyBatis
  - JPQL 등
- JPA에서 사용하는 플랫폼 독립적인 쿼리 언어이다.
  - 사용하는 DB에 상관없이 다 같은 문법을 사용한다.
